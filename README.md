# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. This project allows two players to play the classic game in the browser with a clean and responsive interface.

## 🚀 Features

- 🎲 Two-player gameplay (X vs O)
- ✅ Win detection
- 🤝 Draw detection
- 🔄 Restart/New Game button
- 📱 Responsive design
- 🎨 Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
